1. CertSentinel කියන්නේ මොකක්ද?

Internet එකේ ඕනෑම website SSL Certificate issue කරද්දී (HTTPS setup කරද්දී), ඒ information public ව Certificate Transparency Logs ලා publish වෙනවා. CertSentinel ඒ logs real-time ලා monitor කරලා phishing/fraud domains automatically detect කරනවා.

Real-world example: Attacker කෙනෙකු fake-login-bank.com create කරලා real bank site ලා mimic කරනවා → users trick කරලා passwords steal කරනවා. CertSentinel ඒ domain certificate issue වෙන ඒ moment ම catch කරනවා.

Step-by-step Data Flow

Step Location Description
1 CertStream (Internet) ලොව ඕනෑ SSL Certificate issue වෙද්දී certificate data publish කරයි
2 hunter.py CertStream WebSocket subscribe කරලා live data stream receive කරයි
3 Filter Engine Domain name "login", "bank", "verify", "secure" keywords scan කරයි
4 hunter.py Suspicious නම් TLD + keyword extract කරලා payload build කරයි
5 WebSocket Broadcast All connected clients ලාට JSON alert push කරයි
6 React (App.jsx) Duplicate filter කරලා state update + charts refresh කරයි
7 Dashboard User real-time charts, alerts, badges දකී

6. Run කරන්නේ කොහොමද?
